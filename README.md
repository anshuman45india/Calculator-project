# 🧮 Calculator Web Application

A simple, responsive, and user-friendly **Calculator Web Application** built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic calculations and demonstrates core web development concepts such as DOM manipulation, event handling, and responsive UI design.

---

## 📌 Project Overview

The Calculator project is designed to provide users with a clean and interactive interface for performing basic mathematical operations. It is a beginner-friendly project that helps developers understand how JavaScript interacts with HTML elements and updates the UI dynamically.

This project focuses on creating a real-world mini application without using any external libraries or frameworks.

---

## 🚀 Features

### ✅ Basic Arithmetic Operations
- Addition (+)
- Subtraction (−)
- Multiplication (×)
- Division (÷)

### ✅ User Interface Features
- Clear (C) button to reset the display
- Delete (⌫) option (if implemented)
- Decimal number support
- Real-time display update
- Clean and minimal design

### ✅ Responsive Design
- Works on desktop, tablet, and mobile devices
- Flexible layout using CSS

### ✅ Interactive Functionality
- Button click events handled using JavaScript
- Dynamic display updates using DOM manipulation
- Error handling for invalid inputs

---

## 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| **HTML5** | Structure and layout of the calculator |
| **CSS3** | Styling, alignment, responsiveness |
| **JavaScript (ES6)** | Calculator logic and interactivity |

---

## ⚙️ How It Works

1. User clicks calculator buttons.
2. JavaScript captures button events.
3. Input values are displayed on the screen.
4. Mathematical expressions are evaluated.
5. Result is shown instantly.

The project uses:
- Event listeners
- String manipulation
- Basic evaluation logic
- DOM updates
